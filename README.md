# Transpeso - Caminhão Munck e Guindaste 🏗️

Site institucional desenvolvido para a **Transpeso**, uma empresa especializada em soluções de caminhão munck, guindastes e içamentos, atuando em Brasília - DF. 

🔗 **Acesso ao site em produção:** [transpesodf.com](https://transpesodf.com)

## 🚀 Tecnologias Utilizadas

* **HTML5:** Estruturação semântica e otimizada.
* **CSS3:** Estilização customizada.
* **JavaScript:** Interações dinâmicas da página.

## ✨ Destaques do Projeto

* **Design Responsivo:** Interface totalmente adaptada para dispositivos móveis, tablets e desktops.
* **Otimização de SEO:** Implementação de meta tags estruturadas para melhor ranqueamento no Google e tags Open Graph para pré-visualização em redes sociais (WhatsApp, LinkedIn, etc).
* **Performance:** Estrutura limpa e direta para garantir um carregamento rápido, hospedada via cPanel.
